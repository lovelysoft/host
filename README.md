## host

host 是一款用来方便便捷获取操作系统信息 & 主机硬件配置的工具，免掉进入一个新的主机后一个个命令查看、查找信息，只需一个命令就能获取所有必要的主机信息。

### 可以查看到的主机信息：

```
+--------------+--------------------------------+
| Hostname     | wangkaideMacBook-Pro-2.local   |
+--------------+--------------------------------+
| OS           | darwin 10.14.3                 |
+--------------+--------------------------------+
| BootTime     | 2019-05-17 11:23:17            |
+--------------+--------------------------------+
| Uptime       | 0 Day 0 Hour 45 Minute 14      |
|              | Second                         |
+--------------+--------------------------------+
| CPU Core     |                              4 |
+--------------+--------------------------------+
| Disk Total   | 233 GB                         |
+--------------+--------------------------------+
| Memory Total | 8 GB                           |
+--------------+--------------------------------+
```
